# Volt Keyboard Shortcuts

Complete reference guide for all keyboard shortcuts in Volt.

## Global Hotkeys

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Ctrl+Space` | Toggle Volt window | Default hotkey (configurable in Settings) |
| `F1` | Show help dialog | Works globally |

## View Control

| Shortcut | Action |
|----------|--------|
| `Ctrl+,` | Open Settings |
| `Ctrl+R` | Reload Volt |
| `Ctrl+Q` | Quit Volt / Close window |
| `Esc` | Close window |

## Input Control

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | Open context menu for selected result |
| `Ctrl+A` | Select all text |
| `Ctrl+C` | Copy selected result path (when no text selected) |
| `Ctrl+V` | Paste into search |
| `Ctrl+Backspace` | Delete word |
| `Tab` | Autocomplete with selected result title |

## Configuration

All shortcuts can be customized in Settings (`Ctrl+,`) â†’ Hotkeys section.

### Default Hotkey Conflict Resolution

If `Ctrl+Space` is already in use by another application:

1. Open Volt Settings (`Ctrl+,`)
2. Go to Hotkeys section
3. Click on "Toggle Window" field
4. Press your desired hotkey combination
5. Save settings

The system will warn you if your chosen hotkey is already registered by another app.

## Tips

1. **Quick Launch**: Use `Alt+1` to `Alt+9` to instantly launch the first 9 results
2. **Background Execution**: Use `Ctrl+Enter` to keep Volt open after launching an app
3. **Admin Mode**: Use `Shift+Enter` on Windows to launch applications as administrator
4. **Autocomplete**: Press `Tab` to fill the search bar with the selected result's title
5. **History Management**: Use `Ctrl+Delete` to remove items from your launch history
