## 📋 Git Workflow

**For each task:**
```bash
# Create feature branch
git checkout -b feat/a11y-wcag-aa  # or feat/perf-benchmark, etc.

# Make changes + test locally
bun run dev      # Frontend
cd src-tauri && cargo check  # Rust if needed

# Commit with clear message
git commit -m "feat(a11y): implement WCAG AA patterns

- Add focus trap to SearchBar
- Implement aria-live announcements
- Fix color contrast issues

Fixes #3"

# Push + create PR
git push origin feat/a11y-wcag-aa

# PR checklist:
# - Tests pass (bun run test + cargo test)
# - No TypeScript errors
# - No clippy warnings
# - Manual testing on Windows + macOS + Linux
```
